# WanderLust – Travel Booking Web Application

## 📖 Overview
WanderLust is a full-stack travel booking web application where users can explore, create, edit, and manage travel listings. The application is built using Node.js, Express.js, MongoDB, and EJS following the MVC architecture.

---

## ✨ Features

- User-friendly travel listing interface
- Create new listings
- Edit existing listings
- Delete listings
- View listing details
- Responsive design
- Server-side rendering using EJS
- MongoDB database integration

## 🚀 Installation

1. Clone the repository
2. Run `npm install`
3. Configure MongoDB
4. Run `npm start`
5. Open http://localhost:3000
